## 三、实验原理

### 1. SOS 莫尔斯电码
- **S**：三短闪（· · ·）  
- **O**：三长闪（— — —）  
- **S**：三短闪（· · ·）  

### 2. 时间定义
| 信号 | 时长 |
|----|----|
| 短闪 | 200 ms |
| 长闪 | 600 ms |
| 字母间隔 | 500 ms |
| 单词间隔 | 2000 ms |

### 3. 状态机设计
使用 `enum` 定义不同状态，通过 `millis()` 控制状态切换，避免 `delay()`。
